Visual Tracking UAV - Mambo Drone

Project Overview and Key Contributions:

  • Engineered a high-performance, low-level flight control algorithm specifically for a Parrot Mambo drone, enabling precise and responsive maneuvers.
  • Integrated an Extended Kalman Filter (EKF) into the flight control system for robust state estimation, fusing data from onboard sensors (e.g., IMU, altimeter, camera).
  • Championed the development and optimization of a red color detection algorithm for visual tracking purposes.
  • Successfully reduced the processing time of the color detection algorithm by 30% through algorithmic improvements and efficient coding practices, significantly enhancing real-time tracking capabilities.
  • Implemented visual servoing techniques to enable the Mambo drone to autonomously track and follow a designated red-colored object.
  • Conducted flight tests to validate the performance of the control algorithms, state estimation, and visual tracking system.
  • Analyzed flight data to identify areas for improvement and further refine system parameters.
